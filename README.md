# SolarForge 配布リポジトリ
SketchUp用 太陽光PV/BESS設計プラグイン SolarForge の公式配布場所です。

## インストール / 更新
1. [Releases](https://github.com/Lostriverdelta/solarforge-dist/releases/latest) から最新の `SolarForge-x.y.z.rbz` をダウンロード
2. SketchUp → 拡張機能マネージャ → 拡張機能をインストール → rbzを選択 → SketchUp再起動
3. v0.2.87以降はプラグイン内の「⬆更新」からワンクリック更新できます
